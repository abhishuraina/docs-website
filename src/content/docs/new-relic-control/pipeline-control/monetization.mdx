The cloud rules API includes monetized features based on the usage of New Relic resources. This allows you to manage data processing costs effectively while optimizing your telemetry data.

* **Cost estimation:** The API provides estimates of the cost associated with each rule, helping you make informed decisions about data processing.
* **Resource usage:** Rules are monetized based on their impact on New Relic's infrastructure, ensuring a fair pricing model aligned with resource consumption.
