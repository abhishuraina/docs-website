## Session ID

The Session ID is a unique identifier that is generated every time any app starts. This ID is used to track the session and provide insights into the user's behavior during the session.

By default, the user sessions page displays the detailed entiry information of the user using the session id. You will be able to view the sessions that are in progress or ongoing for a specific application.
