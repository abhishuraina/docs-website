User sessions for Streaming Video and Ads provide insights into the all the sessions that are in progress or ongoing for a specific application. This will provide you with the detailed information about the user sessions, such as:



    * **Session ID**: The unique identifier for the session.
    * **User ID**: The unique identifier for the user.
    * **Application Name**: The name of the application that was entered at the time of installation.
    * **Content Title**: The title of the content that was viewed.
    * **Session Start**: The time at which the session started.
    * **Session End**: The time at which the session ended.
    * **Platform**: The type of device used by the user.
    * **Error Details**: The errors that occurred during the session.
    * **Content Details**: The details of the content that was viewed.
