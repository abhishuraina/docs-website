### Support for Pipeline Control Gateway 1.0.0

New Relic Pipeline Control is the data management component of New Relic Control that helps filter and discard low-value data prior to database ingestion. It consists of an agent called the Pipeline Control gateway that's deployed in your cluster. You create NRQL-based rules to discard unwanted data before it exists your environment.

To know more about Pipeline Control, refer to [the Pipeline Control documentation](/docs/new-relic-control/pipeline-control/overview/).
