### New doc

* Added a new document for [New Relic AI summary for Session Replay](/docs/browser/browser-monitoring/browser-pro-features/session-replay/nrai-summary) to help summarize and map session replay event trail data, enabling a better understanding of errors and successes in your user's browser journey.

### Major changes

* Integrated Amazon CloudWatch Metric Streams with [Amazon Managed Kafka (MSK)](/docs/opentelemetry/best-practices/opentelemetry-best-practices-resources/#amazon-managed-kafka-msk) to generate MSK entities in New Relic. These entities are tagged with `aws.clusterName` or `aws.kafka.ClusterName`, `aws.awsRegion` or `aws.region`, and `aws.topic` or `aws.kafka.Topic`.

### Minor change

* Updated the Windows option for [instrumenting your Azure Functions with the New Relic .NET Agent](/docs/serverless-function-monitoring/azure-function-monitoring/install-serverless-azure-monitoring/#instrument-your-azure-functions-with-the-new-relic-net-agent).
