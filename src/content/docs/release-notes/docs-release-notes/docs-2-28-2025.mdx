### New doc

* [Response intelligence with New Relic AI](/docs/alerts/incident-management/response-intelligence-ai) reduces the time to resolve issues and incidents by using New Relic AI with response intelligence.
* [Browser logs](/docs/browser/browser-monitoring/browser-pro-features/browser-logs/overview) feature captures real-time browser console logs to enhance observability, simplify debugging, and improve user experience.
* [Introduction to Transaction 360](/docs/apm/transactions/workload-performance-monitoring/introdution) offers a revolutionary approach to application monitoring and troubleshooting by automatically capturing and correlating all entities and data related to a specific problem or question.
* The [AWS logs in Context](/docs/logs/logs-context/aws-logs-in-context) offers users a comprehensive, integrated log management solution that enhances both troubleshooting efficiency and system observability within AWS environments.
