### Release note

Stay up-to-date on our most recent release:

* [Fluent Bit v3.2.7](/docs/release-notes/fluentbit-release-notes/fluentbit-27-02-20/)
    * Added support for Flient Bit 3.2.7

* [Logs v250217](/docs/release-notes/logs-release-notes/logs-25-02-17/)
    * New Relic now supports complex data structures in OTLP Logs.

* [Infrastructure agent v1.62.0](/docs/release-notes/infrastructure-release-notes/infrastructure-agent-release-notes/new-relic-infrastructure-agent-1620/)
    * A new version of the Fluent Bit version has been released with a fix for `CVE-2024-50608`, `CVE-2024-50609`

* [Go agent v3.37.0](/docs/release-notes/agent-release-notes/go-release-notes/go-agent-3-37-0/)
    * Implemented a new approach to integrating New Relic with SLOG that is more lightweight, out of the way, and collects richer data.
    * Custom application tags (labels) may be added to all forwarded log events.
    * Added memory allocation limit detection/response mechanism to facilitate calling custom functions to perform application-specific resource management functionality, report custom metrics or events, or take other appropriate actions, in response to rising heap memory size.

* [Node.js agent v12.15.0](/docs/release-notes/agent-release-notes/nodejs-release-notes/node-agent-12-15-0/)
    * Allowed undici error reporting to be disabled with feature flag `undici_error_tracking`
    * Added ability to propagate traceparent and tracestate on incoming server/consumer spans and outgoing client http and producer spans.
    * Added server span transaction naming fallback to the url.path
    * Updated Context class to ensure bi-directional context propagation with opentelemetry bridge.
    * Updated compatibility report.

* [Browser agent v1.284.0](/docs/release-notes/new-relic-browser-release-notes/browser-agent-release-notes/browser-agent-v1.284.0/)
    * Removed FID, replace first interaction detection with INP.
    * Deduplicate prefixes on unhandled promise rejection messages.
    * Upgrade rrweb to 18.

* [.NET agent v10.38.0](/docs/release-notes/agent-release-notes/net-release-notes/net-agent-10-38-0/)
    * Added support for PackageReference with NewRelic.Azure.WebSites packages.
    * Azure Function instrumentation is enabled by default.
