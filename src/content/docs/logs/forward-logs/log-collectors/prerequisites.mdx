Before you install and configure Fluent Bit for logging with New Relic, ensure you have the following prerequisites:
- Access to a Kubernetes cluster
- The necessary tools and permissions for installation and configuration
- Network settings and security policies to allow communication with New Relic endpoints
- A New Relic account
- New Relic Control setup, if applicable

<Callout variant="important">
  You can also manage Fluent Bit via New Relic Control.  For information, see [New Relic Control]().
</Callout>

For information on the first four requirements, see Forward your logs to New Relic.
For information on New Relic Control setup, see Getting started with New Relic Control.
By ensuring these prerequisites are met, you can facilitate a successful deployment of the Fluent Bit logging solution and capitalize on its capabilities within your Kubernetes environment.
