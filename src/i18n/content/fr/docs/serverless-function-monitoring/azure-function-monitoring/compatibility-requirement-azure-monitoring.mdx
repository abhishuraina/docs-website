Avant d’ [instrumenter Azure Functions](/docs/serverless-function-monitoring/aws-lambda-monitoring/instrument-lambda-function/configure-serverless-aws-monitoring), assurez-vous que votre système répond aux exigences suivantes.

## Exigences de base [#requirements]

* Azure Functions hébergé sur [un plan de consommation](https://learn.microsoft.com/en-us/azure/azure-functions/consumption-plan) ou [un plan dédié](https://learn.microsoft.com/en-us/azure/azure-functions/dedicated-plan).

* Vous devez disposer de l&apos;[autorisation requise pour accéder au service kudu](https://learn.microsoft.com/en-us/azure/app-service/resources-kudu#access-kudu-for-your-app).

* Un compte New Relic avec un [rôle d&apos;administrateur](/docs/accounts/original-accounts-billing/original-users-roles/users-roles-original-user-model/#roles) ou un [rôle de gestionnaire d&apos;infrastructure](/docs/accounts/original-accounts-billing/original-users-roles/users-roles-original-user-model#add-on).

* Un <InlinePopover type="userKey" />.

* Agent .NET version 10.30 ou supérieure.

## Supported triggers and Distributed Tracing

You can instrument all trigger types, however Distributed Tracing is supported only in the following cases:

* .NET Stack:

  * Isolated model:
    * HttpTrigger
      * Distributed trace headers will be read from the inbound HTTP request.

  * In-process model (.NET agent version 10.38.0 and later):

    * HttpTrigger
      * Distributed tracing is supported when the function method has a parameter of type `Microsoft.AspNetCore.Http.DefaultHttpRequest`, `Microsoft.AspNetCore.Http.HttpRequest` or `System.Net.Http.HttpRequest`.

    * ServiceBusTrigger:

      * Incoming distributed trace headers will be read if the function method has a parameter of type `Azure.Messaging.ServiceBus.ServiceBusReceivedMessage`. Trace headers are expected to be in the `ApplicationProperties` collection of that object.
      * Distributed trace headers will be inserted into the `ApplicationProperties` collection of outbound Service Bus messages if the function method has a return type of `Azure.Messaging.ServiceBus.ServiceBusMessage` or `Task<Azure.Messaging.ServiceBus.ServiceBusMessage>`.
