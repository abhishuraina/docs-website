New Relic AI Knowledge connector supports 1-click integrations with data sources such as Confluence.

## Connector Configurations

뉴럴릭 AI를 통해 Confluence 콘텐츠 인덱싱을 시작하고 지식 커넥터의 이점을 활용하려면 다음 단계를 따르세요.

<Steps>
  <Step>
    <DNT>[**one.newrelic.com**](https://one.newrelic.com) &gt; **Integrations &amp; Agents**</DNT> 으로 이동합니다.
  </Step>

  <Step>
    <DNT>**NRAI Knowledge Connectors**</DNT> 을(를) 검색하세요.
  </Step>

  <Step>
    <DNT>**Confluence NRAI Knowledge Connector**</DNT> 선택합니다.
  </Step>

  <Step>
    Enter the connector details such as:

    <table>
      <thead>
        <tr>
          <th>
            분야 명
          </th>

          <th>
            설명
          </th>
        </tr>
      </thead>

      <tbody>
        <tr>
          <td>
            Connector name
          </td>

          <td>
            Enter a unique name for your connector, example <DNT>**Demo connector**</DNT>.
          </td>
        </tr>

        <tr>
          <td>
            Connector description
          </td>

          <td>
            Enter a description for your connector.
          </td>
        </tr>

        <tr>
          <td>
            Knowledge category
          </td>

          <td>
            Choose the knowledge category from the drop-down list, this will help New Relic AI to look for information in the right place. You can choose from the following options:

            * <DNT>**Runbook**</DNT>
            * <DNT>**Retrodoc**</DNT>
          </td>
        </tr>
      </tbody>
    </table>
  </Step>

  <Step>
    <DNT>**Continue**</DNT> 을(를) 클릭합니다.
  </Step>

  <Step>
    On the <DNT>**Data source authentication**</DNT> page, select the required authentication method. You can choose from the following options:

    * <DNT>**Basic authentication**</DNT>
    * <DNT>**OAuth 2.0 authentication**</DNT>
  </Step>

  <Step>
    On the Configure data source page, enter the required information and define which documents need to be fetched at what frequency. Click <DNT>**Create**</DNT>.
  </Step>

  <Step>
    On successful configuration, you’ll see the status of your connector on the Connector overview page.

    <table>
      <thead>
        <tr>
          <th>
            분야 명
          </th>

          <th>
            설명
          </th>
        </tr>
      </thead>

      <tbody>
        <tr>
          <td>
            상태
          </td>

          <td>
            Shows you if the data source is available to New Relic AI
          </td>
        </tr>

        <tr>
          <td>
            Last Synced
          </td>

          <td>
            Shows when the data was last updated
          </td>
        </tr>

        <tr>
          <td>
            Syncing
          </td>

          <td>
            If this option is turned off, no new data updates will occur. Existing data will remain unchanged.
          </td>
        </tr>

        <tr>
          <td>
            Delete Connector
          </td>

          <td>
            Deleting a connector will delete all indexed data.
          </td>
        </tr>
      </tbody>
    </table>
  </Step>
</Steps>
