The New Relic tool for Google Gemini Code Assist extension provides intelligent insights and guidance tailored to optimize application performance, monitor infrastructure, and manage deployments—all seamlessly integrated within Gemini Code Assist.

This guide will walk you through the process of installing and using the New Relic tool within the Gemini Code Assist.

## Enable the New Relic tool in Gemini Code Assist [#enable-new-relic-tool]

### Requisitos previos [#prerequisites]

Antes de comenzar, cerciorar de que se cumplan los siguientes requisitos previos para garantizar un proceso de configuración sin problemas:

* An integrated development environment (IDE) is installed on your computer.
* Tienes acceso a Google Gemini Code Assist.
* Gemini Code Assist está instalado en su IDE. Para obtener más información, consulte [la documentación de Google Gemini Code Assist](https://cloud.google.com/gemini/docs/codeassist/overview).

<Callout variant="tip" title="consejo">
  Si desea restringir el acceso a Gemini Code Assist para ciertos roles, configure los ajustes en Gemini Code Assist. Para restringir el acceso a New Relic AI para roles específicos o empleados, puede usar el Control de acceso basado en roles (RBAC) de New Relic. Para obtener más información, consulte [Administración de usuariosNew Relic ](/docs/accounts/accounts-billing/new-relic-one-user-management/user-management-concepts/#understand-concepts).
</Callout>

To enable the New Relic took:

1. From your IDE, log in to [Gemini Code Assist](https://codeassist.google.com) by clicking Sign in with Google.

2. After you are successfully authenticated, return to your IDE.

3. In the <DNT>Gemini Code Assist Chat</DNT> (chat), enter `@newrelic` and press enter. You&apos;ll be prompted to enable the New Relic tool.

4. In the prompt, click the link to enable the New Relic tool. A browser window opens in which you&apos;ll be prompted to enable the New Relic tool within Gemini Code Assist.

5. Enable the New Relic tool, and then click **Connect**.

6. Log in using your New Relic account details. This successfully connects your New Relic account to Gemini Code Assist.

   Now, you can use New Relic tool within the Gemini Code Assist chat extension in your IDE.

## Interact with the New Relic tool [#new-relic-tool]

The New Relic tool within Gemini Code Assist can assist with a variety of tasks and offer valuable insights. To interact with the New Relic tool, in the chat input box, enter your prompt as follows:

`@newrelic <your message>`

Where, `<your message>` is the action you want the tool to execute. For example, `@newrelic What is apdex?`
