      1. To install the browser agent follow [Install the browser agent](/docs/browser/browser-monitoring/installation/install-browser-monitoring-agent) for more information.

      2. Once the browser agent is installed, you will be redirected to the **Installation media streaming agent** page.
      
      3. Click **Continue**.
            
      4. Integrate video monitoring into your application by copying the following code snippet and pasting it immediately below the existing browser agent code to start monitoring your video player.
