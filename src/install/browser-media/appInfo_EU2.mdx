<Callout variant="tip">
    Add our streaming agent to an application that already has our browser agent installed. If you are a first time user and want to add the video agent to a new application, see [Instrument a new browser application](/install/browser-media/?task=newapplication).
</Callout>

<CollapserGroup>
    <Collapser
        id="shaka-player"
        title="Shaka"
      >
      <table>
        <thead>
          <tr>
            <th style={{ width: "200px" }}>
              Install using npm
            </th>
      
            <th>
              Place a JavaScript snippet in frontend code
            </th>
          </tr>
        </thead>
      
        <tbody>
          <tr>
            <td>
            Once you select the **Install using npm** option, you will be redirected to [New Relic Shaka Tracker Agent](https://www.npmjs.com/package/@newrelic/video-shaka/)
            </td>

            <td>

                1. Select the Place a JavaScript snippet in frontend code option.
                2. Click **Continue**.
                3. Install the latest release with all of the newest features and improvements in our latest production-ready SDK.

                ```JavaScript
                curl -L -o newrelic-video-shaka.latest.min.js https://download.newrelic.com/media-agents/browser/shaka/newrelic-video-shaka.latest.min.js
                ```

                4. Include the downloaded script in your application, ensuring it is placed below the existing browser agent code and add the following code snippet to your application to start monitoring your video player:

                ```javascript
                // var player = shaka('player');
                player.version = shaka.Player.version
                const tracker = new ShakaTracker(player);
                ```
                5. Click **See your data**.
            </td>
          </tr>
        </tbody>
      </table>
    </Collapser>
