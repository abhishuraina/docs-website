headingText: Install video agent for an existing browser application
---

<Callout variant="important">
    If you are a first time user and want to add the streaming agent to a new application, see [Install video agent for a new browser application](?task=newapplication).
</Callout>

1. Select your application from the drop-down list, ensure the application has streaming data.

2. Click <DNT>**Continue**</DNT>.

3. Copy the code and insert the snippet as close to the top of your `<head>` tag as possible, but after any position-sensitive META tags (such as X-UA-Compatible and charset).

4. Click <DNT>**See your data**</DNT> to generate the data.
