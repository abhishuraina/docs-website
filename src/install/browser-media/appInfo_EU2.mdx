<Callout variant="important">

</Callout>

    2. Add the following code snippet to your application to start monitoring your video player.

        * Shaka player:

        ```json
        // var player = shaka('player');

        nrvideo.Core.addTracker(new nrvideo.ShakaTracker(player));
        ```
