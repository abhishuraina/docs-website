Add our streaming agent to an application that already has our browser agent installed.

<Callout variant="important">
    If you are a first time user and want to add the media agent to a new application, see [Install video agent for a new browser application](?task=newapplication).
</Callout>

    1. Copy the below code and add the video agent to your application:

    <Callout variant="important">
    Include the downloaded script in your application, ensure it is placed below the existing browser agent code.
    </Callout>

    2. Add the following code snippet to your application to start monitoring your video player.

        <table>
        <thead>
            <tr>

    3. Click **See your data** to generate the data.

<Callout variant="important">
    It may take a few minutes to interact with your application and generate data.
</Callout>
