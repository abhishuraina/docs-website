---
headingText: Install the agent
componentType: default
---

    1. Go to <DNT>**[one.newrelic.com](https://one.newrelic.com) > Integrations & Agents</DNT>.
    
    2. Search and select the **Streaming Video & Ads** tile to open.

    3. Give a name for your application, and click **Create**. 
    
    4. Once the application name is created, click **Save and Continue**.

    5. Select **Mobile**, and click **Continue**.

    6. Select **iOS** to monitor video players in your applications, and click **Continue**.
