---
componentType: default
headingText: Instrument a new mobile application
---

1. Give a name for your application, and click **Create**. Once the application name is created, click **Save and Continue**.

2. Select the Android agent plugin for your project. Regarding the location of the plugin, you have two options:

<Tabs>
    <TabsBar>
        <TabsBarItem id="xcode">
            Xcode
        </TabsBarItem>
        <TabsBarItem id="cocoapods">
            CocoaPods
        </TabsBarItem>
        <TabsBarItem id="swift package manager">
            Swift Package Manager
        </TabsBarItem>
    </TabsBar>

    <TabsPages>
        <TabsPageItem id="xcode">
            1. Download and unzip the New Relic XCFramework SDK.

                <Callout variant="important">
                    Download the latest iOS agent from our [iOS agent release notes](/docs.newrelic.com/docs/release-notes/mobile-release-notes/ios-release-notes).
                </Callout>

            2. Unzip the SDK download.

            3. Drag the `NewRelicAgent.xcframework` folder from the finder into your Xcode project (dropping it onto your Targets Frameworks pane).

            4. Select **Embed & Sign** under the embed column.

            5. Click **Continue**.

            6. Choose one of the following to Integrate and start the agent:

            <Tabs>
                <TabsBar>
                    <TabsBarItem id="objective-c-first">
                        Objective-C
                    </TabsBarItem>
                    <TabsBarItem id="swift-first">
                        Swift
                    </TabsBarItem>
                </TabsBar>

                <TabsPages>
                    <TabsPageItem id="objective-c-first">
                        1. Update your `AppDelegate.m` file.

                        ```java
                            #import <NewRelic/NewRelic.h>
                        ```

                        2. In your `AppDelegate.m` file add this call as the first line of `application:didFinishLaunchingWithOptions`:

                        ```java
                        [NewRelic startWithApplicationToken:@"AA84d7f4ef906a27097190a115fd0e0d3ec3760b78-NRMA"];
                        ```

                        3. Click **Continue**.

                        4. In XCode, select your project in the navigator, and click on the application target.
                        
                        5. Select **Build Phases**, then add a new run script build phase. In the script text area (beneath the Shell line) enter the below script and click **Continue**:

                        ```bash
                        if [ ${CONFIGURATION} = "Debug" ]; then
                            echo "Skipping DSYM upload CONFIGURATION: ${CONFIGURATION}"
                            exit 0
                        fi
                        ARTIFACT_DIR="${BUILD_DIR%Build/*}"
                        SCRIPT=`/usr/bin/find "${SRCROOT}" "${ARTIFACT_DIR}" -type f -name run-symbol-tool | head -n 1`
                        /bin/sh "${SCRIPT}" "AA84d7f4ef906a27097190a115fd0e0d3ec3760b78-NRMA"
                        ```

                        6. On the build and run your application page, click **Continue**.

                        7. Now that you’ve installed New Relic in your app, within minutes you will see data in your New Relic account.
                    </TabsPageItem>

                    <TabsPageItem id="swift-first">
                       1. Update your `AppDelegate.m` file.


                        2. In your `AppDelegate.swift` file add this call as the first line of `application:didFinishLaunchingWithOptions`:

                    </TabsPageItem>

        <TabsPageItem id="cocoapods">
            1. Add the below line to your Podfile:

            ```ruby
            pod 'NewRelicAgent'
            ```

            2. Close your project in Xcode and update by running this command from the terminal in your project directory.

                ### Terminal command

                ```json
                pod install
                ```

            3. Open your project in Xcode by running this command from the terminal in your project directory.

                ### Terminal command

                ```json
                open App.xcworkspace
                ```

            4. Click **Continue**.

            5. Choose one of the options to Integrate and start the agent:
