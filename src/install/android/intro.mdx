Media agent for Android, monitors your mobile app, giving you a comprehensive view of your app's performance. It works for Android apps written using Java or Kotlin.

## Prerequisites [#requirements]

Before you install the Android agent, ensure you have:

- A New Relic [account ID](/docs/accounts/accounts-billing/account-setup/account-id).
- Reviewed the [compatibility and requirements](/docs/streaming-video-&-ads/installation/mobile/android/android-compatibility-and-requirement).

## (Recommended) Guided installation [#guided-install]

To install the Android agent, follow our [guided install](https://onenr.io/08jqaJD9pjl), located directly in the UI.
